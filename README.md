# 📂 File Organizer

A simple Python automation tool that automatically organizes files in a folder by sorting them into categorized folders based on their file extensions.

This tool helps clean messy directories like **Downloads, Desktop, or project folders** by grouping files such as images, documents, videos, and audio into separate folders.

---

## 🚀 Features

* Automatically organizes files by extension
* Supports multiple file categories
* Helps keep folders clean and structured
* Lightweight and fast
* Beginner-friendly Python automation project

---

## 📁 Supported File Categories

The program organizes files into folders such as:

* **Images** → `.jpg`, `.png`, `.jpeg`, `.gif`
* **Documents** → `.pdf`, `.docx`, `.xlsx`, `.pptx`
* **Text Files** → `.txt`
* **Audio** → `.mp3`, `.wav`
* **Videos** → `.mp4`, `.mkv`
* **Code Files** → `.py`, `.js`, `.html`, `.css`
